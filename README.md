# Memory Matching Game

## Overview
This project is a browser-based Memory Matching Game developed using JavaScript. The game challenges users to flip cards and find matching pairs while tracking their performance.

## Features
- Interactive card flipping mechanism
- Matching logic with real-time feedback
- Score tracking system
- Game completion detection
- Responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. The game displays a grid of cards placed face down.
2. The user clicks on two cards to flip them.
3. If the cards match, they remain visible.
4. If they do not match, they flip back after a short delay.
5. The game continues until all pairs are matched.

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Run the index.html file in any browser.
